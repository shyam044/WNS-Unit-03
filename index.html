1)Security Threats Faced by Wireless Devices

  • i)Physical Loss:
    • Devices like laptops, smartphones, etc can be lost or stolen easily
    • if the devices were stolen then the sensitive information could fall into the wrong hands
  • ii)Information Leakage:
    • Sensitive Information can be leaked through careless actions either electronically or physically
  • iii)Using WEP Encryption:
    • WEP stands for Wired Equivalent Privacy
    • Attacker can easily launch attacks on a system
    • Upgrade WEP to WPA/WPA2 to improve security
  • iv)Eavesdropping:
    • Even if your data is Encrypted, attackers can still find a way to look for patterns
    • They don't see your actual content but observes your activities
  • v)Man-in-the-middle attack:
    • Here, the attacker secretly intercepts the communication between user and the Internet

2)Information Leakage:

  • Sensitive Information can be leaked through careless actions either electronically or physically
  • Causes of Information Leakage:
    • i)Weak Access Control:
      • unauthorized users can easily access the sensitive information because of poor permissions

5)FDMA vs TDMA vs CDMA

  • FDMA:
    • FDMA stands for Frequency Division Multiple Access
    • Each User gets a Separate frequency band
    • Divides the channel by Frequency
    • Synchronization is not required
    • It only needs guard bands
    • It is very simple to implement
    • It is less Flexible and Low Data Rate
  • TDMA:
    • TDMA stands for Time Division Multiple Access
    • Each User gets a different time slot
    • Divides the channel by Time
    • Synchronization is required
    • It only needs guard time
    • It is simple to implement
    • It is medium Flexible and Moderate Data Rate
  • CDMA:
    • CDMA stands for Code Division Multiple Access
    • Each User gets a Unique Code
    • Divides the channel by Code
    • Synchronization is required
    • It needs both guard bands and guard time
    • It is complex to implement
    • It is High Flexible and High Data Rate

6)Spread Spectrum:

  • Spread Spectrum is a Communication Technique used to spread the signals over a Frequency bandwidth
  • Spread Spectrum is a WideBand Modulation Technique
  • Principles:
    • i)WideBand Modulation:
      • The Modulated signal is larger than the Original Data Signal
    • ii)Redundancy:
      • Each station allocated a Frequency Bandwidth larger than required to allow redundancy
    • iii)Spreading:
      • Spreading occurs after the signal is generated by using PN-code
      • PN-code stands for Psuedo Noise Code
      • Transmitter and receiver both uses same PN-Code for synchronizatoin
  • Types of Spread spectrum:
    • There are mainly two types:
      • a)FHSS
      • b)DSSS
    • a)FHSS:
      • FHSS stands for Frequency Hopping Spread Spectrum
      • Rapidly Hops between different Frequency Bandwidth
      • It uses Hopping for spreading the data
      • Transmitter and Reciever both uses same Hopping Pattern for Synchronization
      • Example:
        • Bluetooth
    • b)DSSS:
      • DSSS stands for Direct Sequence Spread Spectrum
      • Spreads the signals by using PN-Code
      • It uses Code Sequence for spreading the data
      • Transmitter and receiver both uses same PN-Code for synchronizatoin
      • Example:
        • WiFi

7)Personal Digital Cellular (PDC):

  • PDC stands for Personal Digital Cellular
  • It is a Digital Cellular telecommunication system developed by Japan in 1990's
  • PDC operates in 800 MHz Frequency Bandwidth
  • PDC is designed to replace the analog systems with improving voice clarity and removing background noise
  • It uses ATC for improving the voice clarity,
  • ATC stands for Adaptive Transform Coding
  • Services provided by PDC:
    • Voice Communicatoin
    • Text-Based Communication(SMS)
    • Email
    • Web Browsing
  • PDC plays a crucial role in Japan's Mobile communication History
  • It acted as a Basic Foundation for other technologies like 3G, 4G, 5G, etc...
